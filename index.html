テンプレートについて

完全幅固定タイプです。全体的な横幅は780px、メインコンテンツ部分480px（余白除く）、サブコンテンツ部分200px（余白除く）です。
メニューの横幅は720pxです。デフォルトでは6つ、ひとつのサイズが120pxになるように設定しています。
メインコンテンツは一般的なバナー（468×60px）が入るようにしました。
サブコンテンツ部分はテキストがたくさん書けるように少し多めにとりました。
スクロールさせるくらいテキストの量を増やさないと下が途中で切れてかっこわるくなるという欠点があります。

テンプレートの横幅を変更する際は、基本的にCSSファイルの｢width:XXXpx;｣と記述してあることを修正していただければ背景画像が自動に調整されます。

メインとサブコンテンツの背景は一枚になっており、実際の画像はテンプレートよりも大きく作成しています。よって、メインやサブのサイズを変更したい場合、

***********************************
#contents {
clear:both;
width:700px;
margin:0 20px;
padding:20px 10px 10px 10px;
background:#fff url("../images/bg_contents.gif") -240px 0 repeat-y;
border-bottom:1px solid #C3C3C3;
}
***********************************

｢background:#fff url("../images/bg_contents.gif") -240px 0 repeat-y;｣の｢-240px｣を適当に調整してください。
